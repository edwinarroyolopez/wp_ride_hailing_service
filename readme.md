## Install

To install run this command ```bundle install```

## Start 

To run  this command ```rackup -p 3000```

# Migrations
To run migrations this command ```sequel -m db/migrations postgres://DB_USER:DB_PASSWORD@DB_HOST/DB_NAME```
OR ```rake db:migrate```
